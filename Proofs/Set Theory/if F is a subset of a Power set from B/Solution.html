Theorem. Suppose F⊆℘(B). Then ∪F⊆B.
Proof:
Let a be arbitrary.
Suppose a∈∪F.
Since a∈∪F, we can choose b0 such that a∈b0 and b0∈F.
Since F⊆℘(B), then b0∈F→b0∈℘(B).
Since b0∈F and b0∈F→b0∈℘(B), then b0∈℘(B).
Since b0∈℘(B), then a∈b0→a∈B.
Since a∈b0 and a∈b0→a∈B, then a∈B.
Therefore a∈∪F→a∈B.
Since a was arbitrary, we can conclude that ∪F⊆B.
